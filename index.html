Morse-Talk (voice for the voiceless)

Construction

Input: IR sensor detects Morse signals (short = dot, long = dash). Processing: ESP32 firmware (morse_to_text.ino) times signals, converts them into Morse strings, and maps them to characters. Display: Characters appear on an LCD screen for immediate feedback. Communication: Text is sent via Bluetooth to a Python program. Output: Python converts text to speech; GSM module can place emergency calls.

Working Principle

User taps signals ‚Üí IR sensor captures ON/OFF durations. Firmware distinguishes dots, dashes, and gaps. Morse sequence is matched to a letter using a lookup table. Letter is shown on the LCD and transmitted to Python. Python speaks the message; GSM module dials if emergency code is triggered.
